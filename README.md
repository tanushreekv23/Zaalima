    Consumer360 Project

About:
This project analyzes retail customer data using RFM (Recency, Frequency, Monetary) analysis.
It helps identify high-value customers and churn-risk customers.

Tools Used:
SQL
Python 
Power BI
Windows Task Scheduler

What This Project Does:
Cleans sales data using SQL
Calculates RFM scores using Python
Segments customers (Champions, At Risk, etc.)
Performs Market Basket Analysis
Shows results in Power BI
Updates automatically every week

How It Works:
Raw Data → SQL → Python → Power BI
